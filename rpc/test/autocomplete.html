✗ Erreur : GenewebAutocomplete is not defined
Démo d'intégration. Cette page simule un formulaire de saisie GeneWeb avec des <input list="..."> + <datalist>. Le script geneweb_autocomplete.js les remplace par le widget 4 colonnes quand le serveur RPC est disponible.
Les index sur le serveur sont des fichiers basename.gz. Le nom utilisé dans les appels RPC est le basename sans extension (ex: HenriT_fnames).
